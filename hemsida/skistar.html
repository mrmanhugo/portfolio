SkiStar — Weather ↔ Performance

Python ETL from weather and finance APIs into a star schema, with a Power BI report that compares winter conditions to returns and revenue.

Stack: Python, SQL, Power BI
Model: Star schema, seasonal time series
Focus: Data quality, simple scenarios

Overview

Goal: understand how winter conditions relate to company performance. Build a repeatable pipeline and a clear report for decisions.

Pipeline

  • Ingest: Open-Meteo for winter metrics, Yahoo Finance for annual returns.
  • Transform: Python ETL to a simple star schema.
  • Visualize: Power BI with measures and filters by resort and year.

Outcome

Interactive report with trends and a simple “mild vs cold winter” scenario. The pipeline can be reused for new seasons with minimal effort.
